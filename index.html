Welcome to the Cyberpunk Character Creator / Metachar / Background generator Pages.

This is where you can find the source code written in Python 2.7.10 with PyQT4 for the various character creators for Cyberpunk v3.

Dependencies for running the Python script directly

  • Python 2.7.10
  • PyQt4
  • xhtml2pdf

Running the scripts manually

The following command will start the character creator.

  • python char.py The following command will start the Character Background generator.
  • python bg_generate.py The following command will start the Meta character creator.
  • python meta_char.py
